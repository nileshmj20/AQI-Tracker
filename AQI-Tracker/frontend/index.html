Feature guide

What every main button does

This guide helps visitors understand the dashboard without guessing. The buttons work the same way; this section only explains their purpose.

User help
ChangeQuick city switcher

Opens a city dropdown. Selecting a city updates the AQI number, location name, weather, alerts, charts, area cards and disaster watch.

SearchFind any location

Search accepts a city, country or ZIP code. Suggestions appear while typing; clicking a suggestion applies accurate coordinates.

Use my locationBrowser-based local view

Asks for location permission and updates the dashboard using your current coordinates. It is useful when users do not know their exact city name.

Refresh nowReload live signals

Checks AQI, weather and disaster feeds again. If a live provider fails, the dashboard stays usable and clearly marks hybrid/demo mode.

AlertsRisk explanation popup

Opens a clear alert message based on current AQI, pollutant, weather and hazard signals, so visitors know what action to take.

Reset globeRestore the default globe

Returns the globe to the original zoom and rotation after a visitor has dragged, tilted or zoomed too far.

LayersCompare visual data

Layer checkboxes show or hide AQI, heatmap, monitoring stations, traffic, roads and terrain. Sliders change overlay visibility.

ShareShare current summary

Copies or opens the device share menu with the selected city’s AQI, condition and main risk summary.

FullscreenPresentation mode

Expands the dashboard for project demos, classroom display or focused monitoring. Click again or press Escape to exit.

Generate reportDownload a location report

Exports a report from the backend when available. If the backend is not running, the dashboard downloads a clean demo text report so the button still works.

Search location

Jump to any city.

Search, use your current location, or click the AQI markers on the globe to update the complete dashboard.

Typing: shows city suggestions when available. Search button: applies the typed location to all cards and charts. Use my location: asks browser permission and uses your current coordinates.

Tip: Choose a suggestion for best accuracy. Pressing Search with a custom name still updates the dashboard using safe fallback coordinates.
